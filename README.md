# File Renamer Python

This project is a Python script that automatically renames files in a folder.

## Features
- Renames multiple files automatically
- Keeps original file extensions
- Simple and fast

## Technologies
- Python
- os module

## How to Use

1. Set the folder path in the script:
ruta = "your/folder/path"

2. Run the script:
python script.py
